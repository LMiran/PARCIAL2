Aplicaciones Web

Son programas que se ejecutan en un servidor y son accesibles a través de un navegador web. Permiten a los usuarios interactuar con la información y realizar tareas en línea.

Frontend

Es la parte de la aplicación web que los usuarios pueden ver e interactuar. Incluye el diseño, la estructura y la funcionalidad del lado del cliente, que se ejecuta en el navegador del usuario.

Tecnologías Descripción
HTML Lenguaje de marcado que se utiliza para estructurar el contenido de la web.
CSS Se utiliza para dar estilo a las páginas web, controlando la presentación y el diseño.
JavaScript Lenguaje de programación que añade interactividad a las páginas web.

Backend

Es la parte de la aplicación que se encarga de la lógica del servidor, la gestión de la base de datos y la interacción con el frontend. No es visible para los usuarios, pero es fundamental para que la aplicación funcione correctamente.

Tecnologías Descripción
Base de Datos Almacena la información que la aplicación necesita (ej. MySQL, PostgreSQL).
Servidor Web Gestiona las solicitudes de los clientes y entrega las respuestas adecuadas (ej. Apache, Nginx).
Lenguajes de Programación Ejemplos: Python (Django, Flask), Java (Spring), Ruby (Ruby on Rails), JavaScript (Node.js).

FullStack

Desarrolladores que trabajan tanto en el frontend como en el backend.
